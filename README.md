📌 Description:

AstroNexus is a web-based final year academic project. This application integrates multiple features to provide an interactive and informative user experience related to environment and space.

The project includes the following key features:
- 🌫️ Air Quality Monitoring: Displays real-time air quality data.
- 🌌 3D Solar System: Interactive visualization of planets and space.
- 🌦️ Climate Dashboard: Shows weather and climate-related information.
- 🎙️ Voice-Based Queries: Allows users to interact using voice commands.
